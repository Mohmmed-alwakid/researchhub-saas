🚀 API Integration

The Template Manager integrates with the following API endpoints:

GET /api/templates - Get all templates
GET /api/templates/:id - Get specific template
POST /api/templates - Create new template
PUT /api/templates/:id - Update template
DELETE /api/templates/:id - Delete template
GET /api/templates?action=categories - Get categories
POST /api/templates/:id?action=duplicate - Duplicate template

Authentication: All write operations require JWT authentication with researcher or admin role.
