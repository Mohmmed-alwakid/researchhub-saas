🔧 Variable System

Template Customization

Our enhanced template system includes variable placeholders that allow users to customize templates with their specific product names, tasks, and company information.

Available Variables

[PRODUCT] Product Name
[TASK] Task Description
[WEBSITE] Website URL
[COMPANY] Company Name
[FEATURE] Feature Name

Example Usage

Template Text:
"Welcome to our [PRODUCT] usability study"
After Customization:
"Welcome to our MyApp usability study"
